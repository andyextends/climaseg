Bem-vindo ao ClimaSeg

O ClimaSeg é um sistema inovador projetado para prever, monitorar e mitigar os impactos de desastres climáticos.

Principais Funcionalidades

  • Monitoramento Climático em Tempo Real
  • Sistema de Alertas e Notificações
  • Mapas Interativos e Visualizações Gráficas
  • Integração com Serviços de Mensagens (WhatsApp, E-mail, SMS)
  • Segurança e Proteção de Dados

Tecnologias Utilizadas

  • Microsoft Azure
  • Machine Learning
  • Banco de Dados na Nuvem
  • Azure IoT Hub
  • Azure Functions
